# Expense Tracker

## 📌 Project Description
This is a simple Python-based Expense Tracker that allows users to record and view daily expenses.

## 🚀 Features
- Add expenses
- View all expenses
- Calculate total spending

## 🛠️ Technologies Used
- Python
- File Handling

## ▶️ How to Run
1. Download the project
2. Run: python main.py
